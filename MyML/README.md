

# Dependencies
## Quantum Clustering
 - NumPy
 - SciKit-Learn
 - bitstring

## GPU SL
 - Numpy
 - Numba
 - NumbaPro (only for Single-Link)

## EAC
 - Numpy
 - Numba
 - SciPy
 - SciPy Numba
 - PyTables (for very large sparse matrices)

# Usage
Make sure SciPy Numba and this repository are in the PYTHONPATH, so that the code is availale from anywhere.
